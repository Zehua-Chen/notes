# Position

```css
.class {
  position: ...;
}
```

## `sticky`

```css
h1 {
  position: -webkit-sticky;
  position: sticky;
  top: 0px;
}
```

- Either one of `top`, `left`, `right`, `bottom` has to be specified if
  `position` is set to sticky
- On Safari, `-webkit-sticky` also has to be specified
