# Mutable

```dart
var name = '';
var String name = '';
```

# Immutable

```dart
final name = '';
final String name = '';
```

# Compile Time Constants

```dart
const name = '';
const String name = '';
```

## Constant Values

```dart
var values = const [];
final values = const [];
const values = []; // same as const []
```
