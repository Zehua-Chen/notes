# `children`

## Android

React provides `ViewGroupManager` which provides

- `addView`
- `addViews`
- `getChildAt`
- `removeViewAt`
- `removeView`
- `removeAllViews`

from where children views are manipulated

## iOS/macOS

React extends `UIView`, `NSView` with

- `insertReactSubview`
- `removeReactSubview`

from where children views are manipulated
