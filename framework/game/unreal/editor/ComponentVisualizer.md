# Overview

1. Subclass from `FComponentVisualizer`
2. Override `DrawVisualization`
3. Use `FPrimitiveDrawInterface` to draw visualizations
4. Register component visualizer in modules: `RegisterComponentVisualizer` and
   `MakeShareable`
