\subsubsection{修饰语}

  \begin{itemize}
    \item \important{Adjective} — 修饰Noun或类Noun;
    \item \important{Adverb} — 修饰非Noun;
    \item \important{Preposition + Noun} — 相当于Adj或Adv;
    \item \important{Conjunction + Clause} — 相当于Adj或Adv;
    \item \important{doing, to do, done} — 修饰Noun或类Noun;
    \item \important{Noun或类Noun} — 修饰Noun或类Noun;
    \item \textbf{Only的限制作用}: 修饰对象数量很少, 限度很低;
    \item \textbf{感情色彩提示}: 情感词 (ex. unfortunately) 有提示作用;
  \end{itemize}
